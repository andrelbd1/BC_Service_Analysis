# BC Service

Description
---------------------------------
Analysis performed on data from BC Services 

Quick Setup
---------------------------------
Create a virtualenv and install the requirements.
````bash
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt
````
